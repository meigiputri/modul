Tentang CSS (Cascading Style Sheets)

CSS adalah bahasa stylesheet yang digunakan untuk merubah tampilan dari dokumen yang tertulis dalam HTML.
CSS akan menjelaskan bagaimana elemen dari HTML tersebut harus ditampilkan di layar.
CSS Selector adalah mekanisme untuk menunjuk satu atau lebih element HTML yang akan diberikan perintah styling.
CSS Selector bisa dalam tag, class atau id.
Gunakan class jika selector tersebut untuk banyak element.
Jika class di panggil di CSS, maka akan berpengaruh ke semua class.
Class cocok untuk elemen yang memiliki desain sama.
Sedangkan ID digunakan untuk elemen yang unik dan memiliki ID yang unik dan gaya yang unik, maka CSS hanya menunjuk pada elemen tersebut saja.
